Glossary Page Template

A single HTML page with a built-in editor for creating a glossary that can be hosted anywhere.

Built-in editor

Includes integrated (single-user) editor UI that saves changes back to the HTML file.

Plain HTML

Represents a glossary as a standard dl element for accessibility and portability.

Responsive

Supports different screen sizes using responsive design and includes dark mode support.

Markdown supported

Write descriptions in either plain text or a Markdown-based syntax.

Examples

  • Python 3.11.2 Glossary
  • Glossary — UK Parliament
  • Canada's Weather and meteorology glossary
  • Open Glossary of Edge Computing

Getting Started

Download glossary.html and open it in a browser.

You can also start with empty.html.
